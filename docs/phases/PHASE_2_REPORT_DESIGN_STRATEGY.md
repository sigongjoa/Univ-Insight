# Phase 2 Report Design Strategy: Progressive Disclosure

## 개요
사용자 경험(UX) 디자인의 '점진적 공개(Progressive Disclosure)' 기법을 적용하여, 고교생 타겟에게 최적화된 리포트를 제공합니다. 기본 내용은 누구나 이해하기 쉽게 보여주되, 관심 있는 사람에게만 깊이 있는 정보로 가는 문(링크)을 열어주는 방식입니다.

## 1. 프롬프트 엔지니어링 수정 전략
AI에게 **"두 가지 층위(Layer)"**로 정보를 생성하라고 명확히 지시해야 합니다.

### 수정 전 (예상)
"교수님의 연구 분야를 분석하고 학생에게 추천해줘."

### 수정 후 (제안)
"너는 고등학생을 위한 진로 멘토야. 다음 규칙에 따라 리포트를 작성해.

1. **설명**: 고등학생이 이해할 수 있는 쉬운 비유와 일상 용어를 사용해. (예: 'Vision Grounding' -> '이미지와 글자 연결하기')
2. **전문성**: 쉬운 설명 뒤에는 정확한 '학술 용어'를 괄호 안에 명기해.
3. **확장성**: 학생이 더 깊게 공부하고 싶을 때 검색해볼 수 있는 '심화 탐구 키워드'나 '참고 자료 제목'을 별도로 표시해."

## 2. 리포트 내용 재구성 예시 (Before & After)

### [Before] 기존 리포트 (현재 상태)
**관심 분야**: Vision과 연결된 언어의 이해 (Language Grounding with Vision)
이러한 연구는 이미지와 자연어가 어떻게 서로 연결되는지, 그리고 이를 통해 정보를 추론하고 처리하는 방법에 초점을 두고 있습니다.

### [After] 개선된 리포트 (사용자 아이디어 적용)
**핵심 기술**: 인공지능의 '눈'과 '입' 연결하기 (전문 용어: Vision-Language Grounding)

**🧐 이게 뭔가요?**
우리가 강아지 사진을 보고 "귀여운 강아지가 잔디에 앉아 있네"라고 말하는 것처럼, 컴퓨터도 사진을 보고 문장으로 설명할 수 있게 만드는 기술입니다. 유영재 교수님 연구실에서는 로봇이 사람의 말을 듣고 주변 물건을 찾아오는 데 이 기술을 사용합니다.

**📚 더 깊이 알아보기 (전문가 자료)**
*   **심화 학습 키워드**: Multimodal Learning, Visual Question Answering (VQA)
*   **추천 논문/자료**: [링크] 유영재 교수님 연구실 홈페이지 - Research 탭 확인하기
*   **관련 기초 지식**: 고교 확률과 통계(조건부 확률), 벡터(기하)

## 3. 기술적 구현 팁 (Python & LLM)
단순히 텍스트만 생성하는 게 아니라 구조적인 데이터를 뽑아내는 것이 유리합니다.

### JSON 포맷으로 출력 요청
LLM에게 리포트 내용을 줄글이 아니라 JSON 형태로 달라고 요청합니다.

```json
{
  "topic_easy": "인공지능의 눈과 입 연결하기",
  "topic_technical": "Vision-Language Grounding",
  "explanation": "사진을 보고...",
  "reference_link": "https://lab-url..." // 또는 검색 키워드
}
```

### PDF 생성 시 하이퍼링크 삽입
파이썬의 PDF 생성 라이브러리(ReportLab, FPDF 등)를 사용할 때, 위 JSON의 `reference_link` 데이터를 받아 실제 클릭 가능한 하이퍼링크로 렌더링합니다.

### 링크의 정확성 주의 (Hallucination)
AI가 논문 링크를 직접 생성하게 하면 없는 링크를 만들어낼(할루시네이션) 확률이 높습니다.

*   **안전한 방법**: 교수님의 공식 랩 홈페이지(Lab Page) 메인 링크만 넣기
*   **대안**: 링크 대신 "구글 스칼라 검색어: [키워드]" 형태로 제공하여 학생이 직접 검색하게 유도 (교육적 효과 및 시스템 안정성 확보)

## 결론
"쉬운 설명 + 전문 지식 링크" 구조로 진행합니다. 프롬프트를 수정해서 '고교생 멘토' 페르소나를 주입하고, '심화 탐구' 섹션을 분리합니다.
